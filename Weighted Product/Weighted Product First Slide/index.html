THE WEIGHTED PRODUCT METHOD

The Weighted Product method is one of the multi-criteria decision-making methods used to help select the best car based on various predetermined criteria. In this method, each alternative (car) is evaluated based on the value of each criterion, which is then calculated by multiplying these values after being raised by the weight of each criterion.
In this method, cars are evaluated by calculating the value of each alternative using a mathematical calculation process. The value of each criterion on the alternative is multiplied by the weight of the criterion concerned. Then, the results of each criterion are multiplied by each other to produce a final score.

In this method, there are five criteria used in calculating the determination of cars.

  • Price
    (Cost)
  • Type
    (Benefit)
  • Production Year
    (Benefit)
  • Capacity
    (Benefit)
  • Fuel Comsumption
    (Benefit)

    • There is a status on each criterion, "Cost" and "Benefit" in decision making. If there is one criterion using the Cost status, then the smaller the weight given, the more favorable it will be for the user and this status focuses on reducing expenses. This Cost status is opposite to Benefit which if the weight given is greater, it will benefit the user in the decision system and focus on increasing benefits.

    The main advantage of the Weighted Product method is its flexibility in accommodating individual preferences. Users can assign different weights to each criterion, allowing for truly personalized and relevant results. In addition, the WP method also ensures that only cars with the best final scores will be recommended. This allows users to focus their attention only on the most suitable options, saving time in the search and decision-making process. With the combination of a sophisticated mathematical approach and a focus on user preferences, the Weighted Product method provides an optimal, transparent, and highly reliable solution for choosing a car.
